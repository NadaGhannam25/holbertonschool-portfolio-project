## Overview

**Dierha** is an Arabic web platform that helps users take control of their digital subscriptions in one unified place.

Most people lose track of what they're subscribed to — and end up paying for services they've forgotten. Dierha solves this by bringing all subscriptions together, sending smart renewal reminders, and delivering clear spending insights — without ever requiring access to a bank account.

>  **No bank account linking. No sensitive data. Just simple, smart subscription management.**

---

## Vision & Purpose

Managing subscriptions across Netflix, Spotify, Adobe, STC, and dozens of other services has become a real pain — especially for Arabic-speaking users who have no localized tool for this.

Dierha was built to change that by:

- Giving users a **single dashboard** for all subscriptions
- Sending **automatic reminders** before renewals sneak up on them
- Providing **spending insights** so users know exactly where their money goes
- Being fully **Arabic-first** with RTL support and a familiar, trusted feel

---

##  Key Features

| Feature | Description |
|---------|-------------|
|  **Authentication** | Secure sign up & login |
|  **Subscription Management** | Add, edit, and delete subscriptions |
|  **Unified Dashboard** | All subscriptions in one place |
|  **Smart Categories** | Entertainment · Work · Education · Health |
|  **Renewal Reminders** | Email alerts 3 days before renewal |
|  **Cancel Button** | One click to the subscription's cancellation page |
|  **Billing Calendar** | Visual monthly payment calendar |
|  **Spending Analysis** | Monthly breakdown by category |
| **PDF Export** | Download a full subscription report |

##  Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite |
| Backend | Node.js + Express |
| Database | PostgreSQL |
| Authentication | JWT + bcrypt |
| Reminders | node-cron + Nodemailer |
| Email Service | Resend |
| Charts | Recharts |
| PDF Generation | jsPDF |
| Testing | Postman |
| Deployment | Vercel (Frontend) · Render (Backend) |

---

## Target Audience

- Arabic-speaking users who manage multiple digital subscriptions
- Anyone who wants to track recurring payments without linking a bank account
- Users looking for a simple, private, and localized alternative to Western tools
